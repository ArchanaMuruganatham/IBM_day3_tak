# Student Management API

This is a simple Express.js CRUD application created as part of an academic task.

## Technologies Used
- Node.js
- Express.js
- JavaScript

## How to Run
1. Install Node.js
2. Open project folder in terminal / PowerShell
3. Run:
   npm install express
4. Start server:
   node index.js
5. Server runs at:
   http://localhost:3000

## API Endpoints
- GET /students
- POST /students
- PUT /students/:id
- DELETE /students/:id
